Room Reservation 

Requirements
Must have xampp or wamp installed

Program Description
This is a mini project on an room reservation system which makes use of database.
It involves the use of stored procedures and triggers in the database.
basic functionalities are create, update and delete of entries.
The GUI shows a list of entries.

Setting up the program
Import the reservation.sql in the database folder into your database.
copy the source code to your htdocs folder
